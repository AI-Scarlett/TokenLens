💡 常见问题

Q: 为什么导入模型后看不到统计数据？

A: 导入模型只是注册了模型信息。要看到统计数据，需要通过代理捕获真实的 API 调用。Trae 的核心数据存储在加密数据库中，无法直接读取历史数据。

Q: 代理会影响 Agent 的正常使用吗？

A: 不会。代理只是透明转发请求，不修改任何内容，仅从响应中提取 token 使用数据。

Q: 如何配置 Agent 使用代理？

A: 启动代理后，点击"配置说明"查看详细步骤。最简单的方式是在系统环境变量中设置 HTTP_PROXY。

Q: 支持哪些 Agent？

A: 代理是通用的，支持所有使用 HTTP/HTTPS 调用 LLM API 的 Agent。包括 Trae、CodeBuddy、Cursor、Claude Code、Windsurf、Continue、Aider、VS Code、Void、PearAI 等。启动网关后点击"检测 Agent"可查看本机所有可配置的 Agent。
